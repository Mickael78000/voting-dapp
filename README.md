## Installation

Instructions on how to install and set up this project:

rustc 1.85.0 (4d91de4e4 2025-02-17)\
cargo 1.85.0 (d73d2caf9 2024-12-31)\
npm 10.9.2

Install:
```sh
curl --proto '=https' --tlsv1.2 -sSfL https://solana-install.solana.workers.dev | bash
```
You'll have automatically:
- anchor-cli 0.30.1
- solana-cli 2.1.14
- Node.js v23.10.0
- Yarn 1.22.22

then install:\
Next.js v14.2.26\
avm 0.31.0

Change the version of Solana-CLI: 
```sh
agave-install init 2.0.25
```

Create a scaffold:
```sh
npx create-solana-dapp
```
## Changes to make inside the voting-dapp

Change in your-project/anchor/Cargo.lock:

This file is automatically @generated by Cargo.\
It is not intended for manual editing.\
version = 4

to

This file is automatically @generated by Cargo.\
It is not intended for manual editing.\
version = 3

Add to your-project/anchor/programs/Cargo.toml:
```sh
[features]
default = []
cpi = ["no-entrypoint"]
no-entrypoint = []
no-idl = []
no-log-ix-name = []
idl-build = ["anchor-lang/idl-build"]
anchor-debug = []
custom-panic = []
custom-heap = []

[lints.rust]
unexpected_cfgs = { level = "allow", priority = 2 }
```
Run in the terminal:
```sh
solana config get -ul
solana-test-validator
```
Test your-project/anchor:
```sh
anchor clean
anchor build
anchor test --skip-local-validator --skip-deploy
```
## To display on a website
For the front-end (with the url for more informations):
```sh
npm add @solana/actions
```
https://solana.com/fr/developers/guides/advanced/actions

In the terminal:
```sh
cd ~/PATH/your-project/
npm run dev
```
And in an another terminal:
```sh
cd ~/PATH/your-project/anchor/
anchor deploy
```
In case the finalization transaction doesn't pass:
```sh
solana-test-validator -r
```
Then go to this url:
```sh
https://dial.to/?action=solana-action:http:/localhost:3000/api/vote
```
